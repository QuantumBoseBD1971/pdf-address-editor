# 📄 PDF Address Editor (Desktop App)

## 🚀 Overview
A desktop tool for visually editing address fields in PDF invoices.

> 🧠 Inspired by 1952. Built on 1971.

---

## 🛠️ Tech Stack
- React (UI)
- Electron (Desktop App)
- Python + PyMuPDF (PDF Processing)

---

## ⚙️ Features
- Upload PDF
- Draw rectangle to select address
- Replace with new address
- Export updated PDF

---

## 🚀 Getting Started

### Install dependencies
npm install

### Run app
npm run electron
